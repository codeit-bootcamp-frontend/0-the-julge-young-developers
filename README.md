## Project Overview

### 개요

급하게 일손이 필요한 자리에 더 많은 시급을 제공해서 아르바이트생을 구할 수 있는 서비스입니다.

### 서비스 주요 기능

- 가게 소개와 아르바이트 구인 공고를 등록하고 관리할 수 있습니다.
- 가게는 지원 현황을 확인하고 승인할 수 있습니다.
- 지원자는 아르바이트 구인 공고를 확인하고 공고에 지원할 수 있습니다.
